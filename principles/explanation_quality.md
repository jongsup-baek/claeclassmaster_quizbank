# 해설 품질 원칙

> 퀴즈 해설(explanation) 작성 시 반드시 따르는 원칙

---

## 1. 교재 노트 기반

- 해설의 근거는 해당 강의의 `## 노트` 섹션이다
- 교재 노트에 있는 설명을 바탕으로 해설을 구성한다
- 교재에 없는 내용(일반 지식, 외부 참고자료)은 넣지 않는다

## 2. 용어 일관성

- 교재에서 사용한 용어를 그대로 사용한다
- 예: 교재가 "단일 드라이버"라 했으면 해설에서도 "단일 드라이버"
- 동의어로 바꾸거나 의역하지 않는다

## 3. 깊이와 분량의 일관성

- 해설은 3개 포인트로 구성한다
- 각 포인트는 1~2문장으로 작성한다
- 포인트 구성 패턴:
  - 첫 번째: 핵심 개념 설명
  - 두 번째: 구체적 예시나 비교
  - 세 번째: 실무 관점 또는 주의사항

## 4. 적합성 검증

- 해설 작성 후, 교재 노트와 대조하여 내용이 일치하는지 확인한다
- 교재 범위를 벗어나는 심화 내용은 배제한다
- 해설 간 깊이 수준이 균일한지 점검한다
